ROOM STYLE

Boardroom Examples, basics and add-ons.

Boardroom examples

Boardroom basics

A standard boardroom setup includes a central conference table with chairs, clean cable access, and a discreet service point. It should look symmetrical, premium and uncluttered.

  • Conference table + aligned chairs
  • Notepads + pencils (as per standard / request)
  • Water service point
  • Power access available on request
  • Signage / agenda if provided

Add-ons

Coffee station

Self-serve coffee point with all essentials, presented neatly.

  • Filter coffee / hot water urn
  • Tea selection
  • Milk (and alternatives if required)
  • Sugar, sweeteners, stirrers
  • Cups, napkins, waste bin

Sandwich station

Elegant food display for light lunches or working breaks.

  • Sandwich / wrap selection (allergens labelled)
  • Small plates + napkins
  • Tongs / serving utensils
  • Chilled display if required

Water station

Dedicated hydration point, tidy and easy to refill.

  • Still + sparkling water
  • Glassware + coasters
  • Optional: lemon / mint / infusion
  • Refill plan at peak times
